Adds the missing import for an unqualified type name that refers to a project type declared in another package.

Offered on a bare type reference (one with no . qualifier) when the name is not a type declared in the current file, is not already imported, and a project type with that name exists in a different, named package. When several packages declare a type with the same name, a chooser lets you pick which one to import.

This is the Alt+Enter counterpart of the auto-import that runs when you accept a type from code completion.
