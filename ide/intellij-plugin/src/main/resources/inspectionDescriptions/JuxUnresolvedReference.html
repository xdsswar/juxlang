Reports a bare value reference whose name is not bound anywhere reachable — a typo, or a usage left dangling after the declaration it pointed at was renamed.

Tuned for zero false positives: a name is reported only when it is lowercase and is not a keyword, primitive, built-in global, an import, a binding anywhere in the file (local, parameter, for / catch / lambda variable, destructuring or pattern), nor any type / method / field declared elsewhere in the project. Files with a wildcard import are skipped entirely.

Member accesses (obj.member) and capitalized type references are left to the language server, which sees the standard library and cross-file members.
