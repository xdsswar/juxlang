Reports a value or type reference whose name is not bound anywhere reachable — a typo, or a usage left dangling after the declaration it pointed at was renamed.

Tuned for zero false positives. A value reference is reported only when it is lowercase and is not a keyword, primitive, built-in global, an import, a binding anywhere in the file (local, parameter, for / catch / lambda variable, destructuring or pattern), nor any symbol declared elsewhere in the project. A type reference is reported when it is none of those, not a type parameter, not a jux.std prelude type (Map, List, Throwable, …), and not a type declared anywhere in the project. Qualified paths (a.b.C) and files with a wildcard import are skipped entirely.

Member accesses (obj.member) are left to the language server, which sees the full standard library and cross-file members.
