Validates the extends clause against Jux's single-inheritance model: a class extends exactly one other class — never an interface, record, or enum (E0423), never a final class (E0420), and never a sealed class that doesn't permit it (E0422). Interfaces may extend only interfaces.

Supertypes resolve project-wide; unresolved (library) names are never flagged. The quick-fix moves an interface from extends to implements.
